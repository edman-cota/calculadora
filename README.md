## Instalación y uso
```bash
  # Clonar el repositorio
  git clone https://github.com/edman-cota/calculadora.git
  
  # Mover a la carpeta del repositorio local
  cd calculadora
  
  # Instalar todos los paquetes para uso local
  yarn
  
  # Correr el servidor localmente
  yarn dev
```

Correr eslint
```bash
  yarn lint
```
